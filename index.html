Achtergrond

Mijn naam is Daniel de Cloet. Ik ben, op het moment van schrijven, 17 jaar oud. Ik heb hiervoor een VWO/Gymnasium diploma behaald. Deze heb ik behaald in een zeer technisch vakkenpakket, of zoals ik het noem, mijn versie van een pret-pakket.

Waarom de opleiding ICT

Ik ben al zeer lange tijd bezig geweest met allerlei onderdelen van de ICT wereld.
Hetgeen wat het meeste tijd opneemt is het zijn van server admin van game servers. Ik speel MineCraft op servers, natuurlijk wel met heel veel modsMods (short for modifications) are anything that changes MineCraft's game content from what it originally was.. Natuurlijk komt er best was bij kijken, naast alles wat in de game zelf moet gebeuren, moet ook de server onderhouden worden, moeten discord bots het doen, moet een admin-only panel toegankelijk zijn voor de wat minder technisch aangelegde server owner, en zo voorts. Dit doe ik met heel erg veel plezier.
Daarnaast heb ik ook zeer veel geprogrammeers in allerlei talen. De taal die ik het allerbeste ken moet LUA zijn. Daarna komen, niet noodzakelijk in volgorde van bekendheid en vertrouwdheid, C#, welke ik heb gedaan in een game genaamd Space Engineers, BASH and PERL, allebei een consequentie van het draaien van Linux op mijn devices, Fedora op mijn laptop en Arch op mijn desktop computer. Tenslotte de web-development talen, JavaScript en PHP, maar ook python, allemaal door projecten voor en met een internationale groep vrienden.

Waarom HBO

Vaak krijg ik toch de vraag "Waarom heb je dan geen WO studie gedaan". Het antwoord daarop is misschien simpeler dan gedacht. Er zit nog al een verschil tussen een WO en HBO studie. Voor een opleiding in de richting ICT is een HBO opleiding beter dan een WO opleiding, in mijn opzicht. Een iets minder theoretische opleiding helpt om meer hands-on ervaring op te bouwen. Al heb ik al best wat ervaring, meer kan nooit kwaad.
